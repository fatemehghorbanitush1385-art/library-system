# 📚 Library Management System

یک سیستم مدیریت کتابخانه ساده و کاربرپسند با **Python** و **Tkinter**.

---

## ✨ ویژگی‌ها

- ✅ **رابط کاربری گرافیکی** (GUI) با Tkinter
- ✅ **سیستم ورود و ثبت‌نام** امن
- ✅ **افزودن، حذف و جستجوی کتاب**
- ✅ **ذخیره‌سازی داده‌ها** با JSON
- ✅ **نمایش تاریخچه** کتاب‌های اضافه‌شده
- ✅ **پشتیبانی فارسی**

---

## 🛠️ نیاز‌مندی‌ها

- **Python 3.8+**
- **Tkinter** (معمولاً همراه Python)

---

## 📥 نحوه نصب

### قدم ۱: پوشه رو Clone کن
```bash
git clone https://github.com/fatemehghorbanitush1385-art/library-system.git
cd library-system
```

### قدم ۲: Virtual Environment بساز (اختیاری)
```bash
python -m venv venv
venv\Scripts\activate
```

### قدم ۳: برنامه رو اجرا کن
```bash
python main_gui.py
```

---

## 🎮 نحوه استفاده

1. **برنامه رو اجرا کن**
2. **حساب جدید ثبت کن** یا **وارد شو**
3. **کتاب اضافه کن** (نام، نویسنده، سال)
4. **جستجو** برای پیدا کردن کتاب
5. **حذف کن** اگه نیاز داری
